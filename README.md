# Slim + Bridgetown + Tailwind + Hot Reloading (NOT HMR)

## Requirements

- Bridgetown
- Yarn
- Ruby

## Getting started

```bash
git clone https://github.com/ParamagicDev/bridgetown-slim-starter
cd bridgetown-slim-starter
bundle install && yarn install
```

### Development

```bash
yarn start
```

### Production

```bash
yarn build
```
